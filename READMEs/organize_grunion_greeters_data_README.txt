
================================
organize_grunion_greeters_data.m
================================

Date Created: August 8, 2018

This script takes data from a single CSV file and outputs organized 15day, monthly data, as well as calculates and plot climatologies and anomalies


Components
———————————
PART 1: Reformat 15 day files
PART 2: Adds NaN rows for months/days/years not present in 15day files
PART 3: Reformat monthly files
PART 4: Add NaN rows for months and years not present in monthly files
PART 5: Calculate monthly climatologies & anomalies
PART 6: Plot monthly anomalies
PART 7: Plot max and mean Walker per year


Parameters
———————————

Input: grunion_greeters data

Output: all_days.csv, monthly_all_months.csv, anomalies.csv, anomaly plot
