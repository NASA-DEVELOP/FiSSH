
==================
opedap_sst_2018.m
==================

Date Created: August 8, 2018

This is a MATLAB script used to process MUR Sea Surface Temperature data. It creates several matrices to hold the original data, climatologies, monthly averages, 15-day averages, bounding box, and the lat/lon of the study area. This script also creates seasonal images and monthly sst movies. 

Parameters
———————————

Input: None, pull from url

Output: lat_lon.mat, everything_sst.mat, final_sst_15_day.mat, bounding_boxes.mat, sst_anomalies.mat
