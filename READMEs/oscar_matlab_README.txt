
==================
oscar_matlab.m
==================

Date Created: August 8, 2018

This is a MATLAB script used to process and organize OSCAR surface current data. It creates several matrices to hold the original data, csvs of data, bounding box, and the lat/lon of the study area. 

Parameters
———————————

Input: None, pull from url

Output: lat_lon.mat, oscar_matrix_u.mat, oscar_matrix_v.mat, bounding_boxes.mat
